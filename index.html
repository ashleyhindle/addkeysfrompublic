How to use

bash <(curl -s https://addkeys.ashleyhindle.com/addkeys) -f /root/.ssh/authorized_keys -u ashleyhindle,baby.driver

Usage

./addkeys
[-v|--verbose]
[-b|--baseurl 'https://github.com/']
-u 'comma,separated,usernames'
-f '/root/.ssh/authorized_keys'

Examples

Type Command
1 user from GitHub to 'root' user ./addkeys -u ashleyhindle
2 users from GitHub to 'root' user ./addkeys -u ashleyhindle,baby.driver
1 user from GitHub to 'sysops' user ./addkeys -f /home/sysops/.ssh/authorized_keys -u ashleyhindle
1 user from private GitLab to 'root' user ./addkeys -b https://gitlab.companyname.com -u ashleyhindle
1 user from private GitLab to 'sysops' user ./addkeys -b https://gitlab.companyname.com -f /home/sysops/.ssh/authorized_keys -u ashleyhindle

Notes

The default 'baseurl' is GitHub.com and the default 'file' /root/.ssh/authorized_keys
